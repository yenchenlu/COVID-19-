# COVID-19-對台灣國旅的影響
 資料來源：蒐集各觀光景點在疫情前後的造訪人次資料、Google評論   
 分析方法：以上述資料比較各觀光景點的成長幅度，並透過分析Google評論，找出影響各景點成長幅度多寡的原因

- https://trainerblade.shinyapps.io/R_group1_final_project/
